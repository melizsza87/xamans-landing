# Xamans Landing Page

Strategic Marketing, AI & Growth Systems

## 📋 Descripción

Landing page premium para Xamans, especializada en estrategia de marketing, automatización e inteligencia artificial aplicada. Diseño elegante con paleta navy + gold y animaciones suaves.

## 🎯 Características

- ✅ Diseño responsive (mobile-first)
- ✅ Tipografía elegante (Cormorant Garamond + Inter)
- ✅ Navegación fluida con scroll suave
- ✅ Logo visual con efecto hover
- ✅ Background pattern premium (patrón de ojos)
- ✅ Secciones de servicios, proceso y enfoque
- ✅ Formulario de contacto funcional (Formspree)
- ✅ Animaciones AOS (Animate On Scroll)
- ✅ Optimizado para SEO (meta tags, Open Graph, Schema JSON)
- ✅ Call-to-action estratégicos

## 🚀 Cómo empezar

1. Clona el repositorio:
```bash
git clone https://github.com/melizsza87/xamans-landing.git
cd xamans-landing
```

2. Abre `index.html` en tu navegador o usa un servidor local:
```bash
# Con Python 3
python -m http.server 8000

# Con Node.js
npx http-server
```

3. Visita `http://localhost:8000`

## ⚙️ Funcionalidades Implementadas

### 1. Logo Visual
- Imagen del logo en el header con efecto hover

### 2. Formulario de Contacto
- Integración Formspree (reemplaza `YOUR_FORM_ID`)
- Campos: nombre, email, mensaje
- Validación de cliente

### 3. Animaciones AOS
- Fade-up en elementos principales
- Slide-in en tarjetas
- Zoom-in en CTA
- Sincronizado con scroll

### 4. SEO Optimizado
- Meta tags completos
- Open Graph para redes sociales
- Schema JSON (Organization)
- Canonical URL

### 5. Background Pattern
- Patrón de ojos en hero y CTA
- Overlay semi-transparente para legibilidad
- Fixed attachment en desktop

## 📧 Configurar Formspree

1. Ve a: https://formspree.io
2. Regístrate gratis
3. Crea un nuevo formulario
4. Copia el `form_id`
5. En `index.html` línea ~150, reemplaza `YOUR_FORM_ID` con tu ID
